## 🚀 Featured Projects

### 🔐 File Integrity Monitor

A cybersecurity-focused tool that monitors file integrity using SHA-256 cryptographic hashing.

**Features**

* Detects file creation, deletion, and modification
* Generates SHA-256 file hashes
* Maintains baseline snapshots for comparison
* Simulates endpoint security monitoring concepts

**Technologies:** Python, Hashlib, File Handling
